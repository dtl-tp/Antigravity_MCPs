# NotebookLM MCP Server

## Descripción
Conecta las capacidades de Antigravity con las libretas de NotebookLM de Google. Ideal para gestionar corpus de información complejos y generar overviews de alta calidad.

## Capacidades
- **Gestión de Notebooks**: Crear, listar y renombrar libretas.
- **Gestión de Fuentes**: Añadir fuentes desde URLs, YouTube, PDFs subidos o texto pegado.
- **Análisis IA**: Consultar fuentes existentes en el notebook para obtener respuestas basadas exclusivamente en esos datos.
- **Generación de Contenido**: Crear resúmenes de audio, presentaciones de diapositivas, quizzes y mapas mentales.

## Ejemplos de Uso
- "Crea un audio overview de esta libreta de investigación"
- "Añade este hilo de documentación a mi notebook de 'Arquitectura'"
- "Genera un quiz basado en las fuentes del proyecto"
