# Next DevTools MCP Server (Vercel)

## Descripción
Servidor MCP oficial de Vercel diseñado para optimizar el ciclo de desarrollo de aplicaciones Next.js 16+. Permite al asistente de IA interactuar con los internos del servidor de desarrollo.

## Requisitos
- **Next.js 16+**: El proyecto debe utilizar la versión 16 o superior.
- **Servidor en Ejecución**: El servidor de desarrollo (`npm run dev`) debe estar activo.

## Capacidades
- **Diagnóstico en Tiempo Real**: Inspección de estados internos del servidor y componentes.
- **Asistencia en Migración**: Herramientas para facilitar el upgrade a versiones modernas de Next.js.
- **Configuración de Cache**: Ayuda para habilitar y optimizar Cache Components.
- **Acceso a Knowledge Base**: Consulta directa de la documentación interna de Next.js.

## Ejemplos de Uso
- "Inspecciona por qué este componente de servidor no está renderizando en Next.js"
- "¿Cómo puedo optimizar el cache de esta ruta según Next DevTools?"
- "Ayúdame a migrar este proyecto a Next.js 16"
