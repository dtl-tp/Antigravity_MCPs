# MarkItDown MCP Server (Microsoft)

## Descripción
Herramienta oficial de Microsoft para convertir diversos formatos de archivo complejos en contenido Markdown estructurado y optimizado para IA.

## Formatos Soportados
- **Microsoft Office**: Word (.docx), Excel (.xlsx), PowerPoint (.pptx).
- **Documentos**: PDFs (con soporte para OCR básico).
- **Multimedia**: Imágenes (extracción de metadatos y texto) y Audio (transcripción).
- **Web/Datos**: HTML, CSV, JSON, XML.

## Capacidades
- **Conversión de Archivos**: Transforma documentos complejos manteniendo la semántica (cabeceras, tablas, listas).
- **Batch Processing**: Capacidad para convertir directorios completos.
- **Optimización para LLM**: El formato Markdown resultante es el lenguaje nativo "preferido" por los modelos de lenguaje.

## Ejemplos de Uso
- "Convierte este PDF técnico a Markdown para que podamos analizarlo"
- "Resume los puntos clave de esta presentación PowerPoint"
- "Extrae los datos de esta tabla Excel y formátalos como Markdown"
