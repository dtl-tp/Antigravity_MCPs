# GitHub MCP Server

## Descripción
Servidor MCP oficial para interactuar con la API de GitHub. Permite realizar operaciones completas en repositorios, issues, pull requests y más.

## Capacidades
- **Gestión de Repositorios**: Crear, buscar y listar repositorios.
- **Gestión de Archivos**: Leer, crear, actualizar y borrar archivos en cualquier rama.
- **Colaboración**: Crear issues, pull requests (y revisarlos), y añadir comentarios.
- **Búsqueda de Código**: Buscar fragmentos de código en todo GitHub.

## Ejemplos de Uso
- "Crea un nuevo issue en el repositorio X diciendo que falta documentación"
- "Prepara un Pull Request con los cambios que acabamos de hacer"
- "Busca en el repo cómo está implementado el componente Button"
