# Supabase MCP Server

## Descripción
Permite gestionar la infraestructura de Supabase directamente desde el chat.

## Capacidades
- **Gestión de Proyectos**: Listar proyectos y organizaciones.
- **Base de Datos**: Gestión de tablas, tipos y esquemas.
- **Querying**: Ejecutar operaciones de base de datos de manera asistida.

## Ejemplos de Uso
- "Listame mis proyectos activos en Supabase"
- "¿Cuál es el esquema de la tabla 'perfiles'?"
