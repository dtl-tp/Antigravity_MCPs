# Context7 MCP Server (Upstash)

## Descripción
Context7 es un servidor MCP diseñado para inyectar documentación técnica actualizada y ejemplos de código directamente en el contexto de la IA. Su objetivo principal es evitar alucinaciones causadas por información obsoleta en el modelo de lenguaje.

## Capacidades
- **Fetch de Documentación**: Obtiene la última versión oficial de documentación para librerías populares.
- **Ejemplos de Código**: Proporciona fragmentos de código precisos y compatibles con las versiones más recientes.
- **Inyección de Contexto**: Permite que el asistente "sepa" sobre actualizaciones técnicas que no estaban en su entrenamiento original.

## Comandos Útiles
- "Usa context7 para revisar la documentación de Next.js"
- "Busca en context7 cómo implementar auth en Supabase"
