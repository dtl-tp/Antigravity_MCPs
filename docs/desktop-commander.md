# Desktop Commander MCP Server (wonderwhy-er)

## Descripción
Un servidor MCP de alta potencia que otorga al asistente de IA un control operativo profundo sobre la máquina local. Transforma el chat en una estación de trabajo activa.

## Capacidades

### Terminal y Procesos
- **Ejecución de Comandos**: Soporte para comandos interactivos y de larga duración.
- **Gestión de Sesiones**: Control de múltiples terminales y procesos background.
- **Code Execution**: Ejecución de Python/Node.js en memoria.

### Sistema de Archivos
- **Búsqueda Ripper**: Integración con `ripgrep` para búsquedas ultrarrápidas de patrones de código.
- **Manipulación de Archivos**: Leer (con streaming para archivos grandes), escribir, mover y editar quirúrgicamente fragmentos de código (diff-style).

### Documentos y Datos
- **Análisis de Excel**: Soporte nativo para lectura y edición de `.xlsx` y `.csv`.
- **Gestión de PDF**: Extracción de texto y creación de PDFs.

## Ejemplos de Uso
- "Busca todas las referencias a 'API_KEY' en el proyecto usando ripgrep"
- "Abre una terminal interactiva y corre los tests unitarios"
- "Analiza la pestaña 'Ventas' del archivo Excel proporcionado"
